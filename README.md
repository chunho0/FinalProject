Flight Reservation System
HOW TO USE

Login Information
Default Admin Account
\nUsername: admin
\nPassword: admin
Notes:
Username is not case-sensitive.
Password is case-sensitive.
The default admin account is all lowercase.


Current Functions

Main Menu
After logging in as admin, the following menu will appear:
1. Add Flight
2. Update Flight
3. Remove Flight
4. Display Flight
5. Search Flight
6. View Booking (Not completed yet)
7. Manage User Account
8. Logout


Flight Functions
Add Flight
The admin can add a new flight by entering:
Flight Number
Departure Airport
Landing Airport
Departure Time
Price
Aircraft Model
The system already contains 3 default demo flights for testing.
Notes:
Departure time must follow the format shown in the console.
Time uses 24-hour format.

After entering all details, the system displays a confirmation preview before adding the flight.

Update Flight
The admin can update a flight by entering the flight number.
Current update options:
1. Departure Airport
2. Landing Airport
3. Departure Time
4. Price
5. Aircraft Model
After updating, the system displays a preview and asks for confirmation before saving the changes.

Remove Flight
The admin can remove a flight by entering the flight number.
The system will display the flight details and ask for confirmation before removing the flight.

Display Flight
This function displays all flights currently stored in the system.

Search Flight
The system currently supports 2 search methods.
1.Search by Flight Number

2.Search directly using the flight number, Route and Date
Search using:
Departure Airport
Landing Airport
Departure Date
If matching flights are found, the system will display all available flights.

User Account Management
The system also includes a User Account Manager.

Current functions:
1. Display Users
2. Add Users
3. Remove Users (Not completed yet)
4. Modify Users (Not completed yet)

Display Users
The system displays all users currently stored in the user list.
Default users include:
1 Admin account
3 Test Guest accounts

Add Users
The admin can choose between:
1. Add Admin
2. Add Guest

Add Admin
Creates a new admin account that can be used for admin login.
Add Guest
Creates a guest account with additional information such as:
Email
Phone Number
Address
Membership Level
Guest login functionality has not been completed yet.
